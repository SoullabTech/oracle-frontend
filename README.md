## Oracle Frontend

This is the frontend for the Spiralogic Oracle System, powered by Vite + React + Tailwind.

### 🧪 Getting Started

```bash
npm install
npm run dev
```

### 🧼 Dev Scripts
- `npm run dev` – Start dev server
- `npm run build` – Production build  
- `npm run lint` – Lint code
- `npm run preview` – Preview prod build

### ⚙️ Requirements
- Node.js >=20.0.0
- npm >=9
